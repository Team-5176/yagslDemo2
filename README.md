# YAGSL Demo 2

YAGSL + Photonvision base template. FRC Team 5176 Robobills.

## Simulation

Run Photonvision in the background. For macOS with Java 17, and the latest release saved in `~/code/photonvision/`, type this into the terminal:

```zsh
$ java -jar ~/code/photonvision/photonvision-dev-v2025.3.1-63-g843c887d-macarm64.jar
```
After that, use the built-in WPILib Simulation tool.
